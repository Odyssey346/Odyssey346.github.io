The Guide to Installing the Config

This config is aimed at new players, therefore it might be confusing to install. Therefore, I've made this guide.

-----------------------------------------------------------------

1. Open Steam if it ain't open.

2. Right click Team Fortress 2.

3. hover your mouse over Manage

4. Select "Browse local files"

5. Go to tf/cfg folder when the local files are open.

6. Drag and drop the autoexec.cfg file into the cfg folder.

Now you're done! Enjoy!
